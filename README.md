## 功能🧩

### 歌词显示✨

根据`标题`+`艺术家`+`唱片集`自动匹配歌词，请确保此三项存在：

![图片描述](./文件属性.png)

### 歌词展开✨

鼠标移入歌词区域，左侧区域显示竖杠，点击展开歌词。

### 专辑图显示✨

自动加载歌曲内置专辑图显示。

### 全屏及浅色/深色切换✨

操作列表上的功能按钮区。
